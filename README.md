# Hyperspectral Band Selection Review (2023–2025)

This repository supports the review paper:

"Review on Different Band Selection Approaches for Classification of Hyperspectral Images"

## Purpose
This repository provides structured resources, dataset references, and methodological summaries used in the review.

## Covered Methods
- Filter-based methods (Entropy, Correlation)
- Wrapper-based methods (SFS, GA)
- Embedded methods (SVM-RFE, CNN, Transformers)
- Hybrid methods (RL, MOO, DWT fusion)

## Benchmark Datasets
- Indian Pines (AVIRIS)
- Salinas (AVIRIS)
- Pavia University (ROSIS-03)
- Kennedy Space Center (AVIRIS)

## Note
This is a review paper. No new dataset or code is introduced.
All datasets and methods are referenced from published literature.
